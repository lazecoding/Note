# 如何自我介绍

自我介绍觉得了面试官对你的第一印象，第一印象太重要了。自我介绍一定要简洁而突出重点，要让面试官能消化掉这段自我介绍的内容，太长会让对方抓不到重点，还啰嗦。自我介绍一定要熟练，滚瓜烂熟，不要因为紧张卡壳。

自我介绍是要解释三个问题：

- 我为什么选择这家公司？
- 我为什么应聘这个职位？
- 聘用我能给公司带来什么？

面试官想从自我介绍中知道：

- 这个人能力层面是否可以胜任这份工作？
- 这个人适合我们公司/团队的文化/行事习惯么？
- 这个人自己有没有想清楚加入我们？

自我介绍注意点：

- 优势、亮点、基本情况。
- 言简意赅、语言精炼，控制时间（30 - 45 S）。
- 和应聘岗位相关的经历。
- 为什么能够胜任。
- 为什么要应聘该岗位，即求职动机，表示愿意长期从事这个岗位。
