# 基础

基于 Java 构造算法模型。

### 常用数学函数

| 方法                  | 作用                  |
| -------------------- | -------------------- |
| Math.abs(a);         | a 的绝对值            |
| Math.max(a, b);      | a 和 b 中的较大者      |   
| Math.min(a, b);      | a 和 b 中的较小者      |
| Math.sin(a);         | a 的正弦函数           |
| Math.cos(a);         | a 的余弦函数           |
| Math.tan(a);         | a 的正切函数           |
| Math.exp(a);         | a 的正弦函数           |
| Math.log(a);         | a 的余弦函数           |
| Math.pow(a, b);      | a 的正切函数           |
| Math.random();       | a 的正切函数           |
| Math.sqrt(a);        | a 的正切函数           |
| Math.E;              | 常数 E                |
| Math.PI;             | 常数 π                |
